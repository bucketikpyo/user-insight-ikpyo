## 🎯 리서치 목적

PV(Page View)가 낮은 롱테일 상품 중 **숨겨진 보석(고매력 저PV)**을 발굴하고,
스타일링샷(SS) 부스팅을 통한 PV 성장 가능성 검증

---

## 📋 주요 발견

1. **PV와 매력도는 무관** (상관계수 r=0.077, p=0.497)
2. **💎 숨겨진 보석(C1)** 9개 상품 발굴 - 고매력이지만 노출 부족
3. **SS→PV 선순환 구조** - ⭐스타 상품은 초반 SS 축적으로 성공
4. **SS 최적 개수** - 2개에서 최대 효율 (+152% UV), 6개 이후 한계효과 급감
