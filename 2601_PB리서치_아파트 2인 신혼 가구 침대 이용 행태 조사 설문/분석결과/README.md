## 💡 신규 패브릭 침대 기획 시사점

### 헤드보드 디자인 핵심 요소

1. **높이 & 각도**
   - 권장: 높이 80-100cm, 각도 5-10도 후경
   - 앉아서 기댈 수 있는 구조

2. **쿠션감 & 지지력**
   - 두툼한 쿠션 + 적절한 지지력 균형
   - 고급 충진재 (메모리폼, 고밀도 우레탄)

3. **구조적 안정성**
   - 장시간 기대어도 안정적인 프레임

4. **모듈형 옵션**
   - 추가 쿠션으로 높이/각도 조절 가능
   - 사용자 맞춤 구조

5. **디자인 차별화**
   - 기능성 + 심미성
   - 고급 패브릭 소재

## 🔧 분석 방법

### 사용 도구
- Python 3.9
- pandas, matplotlib, numpy

### 분석 스크립트
- `survey_analysis.py` - 1-3단계 (데이터 로드, 정제, 그룹 정의)
- `survey_analysis_part2.py` - 4-6단계 (핵심 분석, 교차 분석, 시각화)

### 주요 처리 로직
1. **설문 로직 처리**
   - Q3 "거의 없다" → Q4/Q5 스킵
   - Q8 "없음(무헤드)" → 무헤드 전용 질문, Q9/Q10/Q11 스킵
   - Q9 불만족 → Q10/Q11 응답

2. **복수 선택 문항 처리**
   - Q5 (활동), Q6 (가구), Q10 (불편점), Q11 (보완방법)
   - 리스트 형태로 변환 후 분석

3. **파생 변수 생성**
   - 타겟 그룹 분류 (2인가구, 아파트, 신혼)
   - 행태 분류 (빈도, 시간, 만족도)
