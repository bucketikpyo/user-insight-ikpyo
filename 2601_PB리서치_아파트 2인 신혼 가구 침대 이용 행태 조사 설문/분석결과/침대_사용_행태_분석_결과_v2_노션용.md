# 침대 사용 행태 설문 분석 결과

## 📊 Executive Summary

*분석 대상: 전체 n=675, 2인가구+아파트 n=234, 신혼타겟 n=131*

### 주요 발견사항

1. **신혼 타겟의 침대 사용 행태** (침대 사용자 n=78 기준)
   - 침대에서 영상 시청: **82.1%** (전체 66.6% 대비 **+15.5%p**)
   - 장시간 사용(1시간 이상): **39.7%**
   - TV 보유 시 영상 시청: **91.2%**

2. **헤드보드 중요도 인식** (신혼타겟 n=131 기준)
   - 신혼 타겟의 **91.6%**가 헤드보드를 중요하게 생각 (매우+어느정도)
   - 전체 86.1% 대비 **+5.5%p** 높음

3. **헤드보드 타입별 만족도 차이** (무헤드 제외, 신혼타겟 n=121 기준)
   - 패브릭/쿠션 타입 만족도: **63.3%**
   - 하드 타입 만족도: **19.4%**
   - **3.3배 차이** 발생

4. **헤드보드 불편 사항** (불만족 응답자 기준)
   - 신혼 타겟 Top 3: 머리·목 지지 부족(47.4%), 각도 불편(44.7%), 쿠션감 부족(44.7%)
   - 보완 방법: 베개 여러 개 받침(84.2%), 쿠션 추가(50.0%)
   - 주관식 응답 n=6: "수납형은 기댈 수 없어요", "원목이라 딱딱해서 불편"

## 6. 결론 및 제언

### 6.1 핵심 인사이트

1. **신혼 타겟은 침대를 단순한 수면 공간이 아닌 생활 공간으로 활용** (n=131)
   - 침대 사용자(n=78) 중 82.1%가 침대에서 영상 시청
   - 39.7%가 1시간 이상 장시간 사용
   - TV 보유 시 91.2%가 영상 시청

2. **헤드보드 중요도 인식이 높으나 만족도는 낮음** (n=131)
   - 91.6%가 헤드보드를 중요하게 생각
   - 하지만 현재 만족도는 37.2% (무헤드 제외 n=121)
   - **갭(Gap)**: 중요도 91.6% vs 만족도 37.2% = **54.4%p 차이**

3. **소프트 타입 헤드보드에 대한 선호도가 압도적** (무헤드 제외 n=121)
   - 소프트 타입 만족도 63.3% vs 하드 타입 19.4%
   - 약 **3.3배 차이**
   - 주관식(n=6): "원목 딱딱해서 불편", "수납형은 기댈 수 없어요"

### 6.2 제품 개발 제언

1. **편안한 기대기 경험에 집중**
   - 머리·목 지지력 강화
   - 각도 조절 가능한 디자인
   - 충분한 쿠션감 제공
   - ⚠️ **주의**: 수납형 헤드보드는 기대기 불편 → 기능 충돌 해결 필요

2. **TV/OTT 시청에 최적화된 디자인**
   - 장시간 사용에도 편안한 구조
   - 시청 각도를 고려한 헤드보드 높이/각도
   - 패브릭/쿠션 소재로 하드 타입 대비 3.3배 높은 만족도 확보

3. **기능과 편안함의 균형**
   - 수납 기능 + 기대기 편안함을 동시에 만족하는 혁신적 구조
   - 틈새/선반 구조 지양
   - 소프트 타입 소재 적용

4. **신혼 타겟 맞춤형 마케팅**
   - "침대에서의 새로운 라이프스타일" 강조
   - TV/OTT 시청 경험 중심의 제품 소구
   - 2인 사용 시나리오 강조
